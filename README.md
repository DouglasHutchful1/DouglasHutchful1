<h3 align="left">About Me</h3>

I am a .NET Full-Stack Developer specializing in backend-driven web applications and structured system design.

My experience centers around building and maintaining business-critical applications using ASP.NET Core and SQL Server. I focus on writing clean, maintainable code and designing systems that are structured, scalable, and easy to extend.

Key strengths:

- Designing modular, maintainable backend architectures  
- Migrating legacy ASP.NET MVC systems to modern .NET Core  
- Implementing structured access control and role-based systems  
- Building secure RESTful APIs with proper separation of concerns  
- Writing optimized SQL queries and stored procedures  
- Integrating reporting systems and business workflow automation  

I approach engineering with a strong emphasis on clarity, long-term maintainability, and real-world problem solving rather than surface-level complexity.
